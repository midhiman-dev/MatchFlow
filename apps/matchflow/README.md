# MatchFlow - Smart Stadium Assistant

MatchFlow is a cricket-aware smart stadium assistant and operations platform. It helps manage fan surges during match moments such as innings breaks and mass exits through real-time crowd intelligence, queue-aware routing, and safety-first guidance.

## Run Locally

**Prerequisites:** Node.js

1.  Navigate to this directory:
    `cd apps/matchflow`
2.  Install dependencies:
    `npm install`
3.  Run the app:
    `npm run dev`
4.  Open your browser at `http://localhost:3000`

## Features

- **Fan App:** Real-time match engagement, queue alerts, and route guidance.
- **Operator Dashboard:** Live congestion heat mapping and emergency controls.
- **Steward View:** Real-time tactical guidance for crowd management.
- **Demo Simulator:** Pre-configured scenarios forinnings breaks, DRS spikes, and exits.
